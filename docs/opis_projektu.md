# Opis Projektu

## 1. Projekt dotyczy aplikacji wspierającej studentów w planowaniu nauki.
## 2. Rozwiązuje problem braku struktury i przeciążenia informacjami.
## 3. Odbiorcami są studenci w wieku 18–25 lat chcący lepiej organizować zadania.
## 4. Funkcje: lista zadań, tygodniowy kalendarz, przypomnienia.
## 5. Dodatkowo proste statystyki postępów i priorytetyzacja zadań.
